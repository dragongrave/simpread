工作流程
====

本章节我们将为大家介绍 Git 的工作流程。

一般工作流程如下：

1.  克隆 Git 资源作为工作目录。
2.  在克隆的资源上添加或修改文件。
3.  如果其他人修改了，你可以更新资源。
4.  在提交前查看修改。
5.  提交修改。
6.  在修改完成后，如果发现错误，可以撤回提交并再次修改并提交。
